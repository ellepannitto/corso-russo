01

Cos'è Anki e installazione

Anki è un'app di flashcard basata sulla ripetizione dilazionata: le carte vengono ripresentate più spesso quando c'è il rischio di dimenticarle e più raramente quando sono ormai consolidate, così lo studio si concentra dove serve davvero.

Android → si cerca AnkiDroid su Google Play. È gratuito.

iPhone → si cerca AnkiMobile Flashcards su App Store. Costa circa 25 €.

Computer (Windows/Mac/Linux) → si scarica Anki Desktop, gratuito, da apps.ankiweb.net.

Esiste anche una versione web gratuita su ankiweb.net, ma è meno comoda da usare ogni giorno.

Solo AnkiWeb dal browser?

Non è possibile importare russo.apkg direttamente da ankiweb.net: serve prima uno dei client (AnkiDroid, AnkiMobile o Anki Desktop) per fare l'import una volta sola. Con Anki Desktop: si installa, si importa il file (passo 03), poi si attiva la sincronizzazione (Anki → Sincronizza, serve un account gratuito su ankiweb.net). Da quel momento le carte compaiono anche su ankiweb.net/study, utilizzabili da browser su qualsiasi computer.

02

Salvare il file

Quando si riceve il file russo.apkg (via email, WhatsApp, link…):

Su telefono

  • Lo si tiene premuto (o si tocca sui tre puntini accanto al file) e lo si salva nella cartella Download.
  • Su iPhone, lo si salva nell'app File (in "Sul mio iPhone" o iCloud Drive).

Su computer

  • Lo si scarica con il bottone in cima alla pagina (o dal link ricevuto) e lo si salva in una cartella facile da ritrovare, ad esempio Download o Desktop.
03

Importare il vocabolario

Su Android (AnkiDroid)

  1. Si apre la cartella Download sul telefono.
  2. Si tocca il file russo.apkg.
  3. Se viene chiesto con quale app aprirlo, si sceglie AnkiDroid.
  4. Compare un messaggio con il numero di carte importate. Si tocca OK.

Su iPhone (AnkiMobile)

  1. Si apre l'app File e si trova il file russo.apkg.
  2. Si tiene premuto il file → si tocca Condividi → si sceglie Copia in AnkiMobile.
  3. Anki si apre e mostra un messaggio di importazione completata. Si tocca OK.
Se qualcosa non funziona

Se non si riesce a "condividere" con AnkiMobile, si può provare da Safari: si apre il link del file direttamente nel browser e Safari gestisce l'apertura automatica.

Su computer (Anki Desktop)

  1. Si scarica russo.apkg con il bottone in cima alla pagina (di solito finisce nella cartella Download).
  2. Si apre Anki Desktop.
  3. Si va su File → Importa file... (oppure si trascina il file russo.apkg sulla finestra di Anki).
  4. Si seleziona russo.apkg e si conferma: le carte vengono aggiunte subito, senza altre schermate.
04

Creare mazzi personalizzati (per lezione o per tema)

Tutte le carte arrivano in un unico mazzo "Russo", ma ogni carta ha delle etichette (tag) che indicano la lezione e il tema — ad esempio lez-3, alfabeto, cibo. Si possono usare per studiare solo un sottoinsieme di parole, senza bisogno di un file diverso per ogni lezione.

  1. In Anki si apre Sfoglia (l'icona della lente/lista carte).
  2. Nella barra di ricerca si scrive un tag, ad esempio: tag:lez-3 oppure tag:cibo.
  3. Si selezionano tutti i risultati (su AnkiDroid: si tocca e si tiene premuto su una carta, poi "Seleziona tutto").
  4. Si sceglie Crea filtro / Crea mazzo filtrato (oppure "Cambia mazzo" per spostarle in un mazzo separato invece che filtrarle).
  5. Si dà un nome al nuovo mazzo (es. "Lezione 3") e si conferma.

Si possono combinare più tag nella ricerca: tag:lez-1 OR tag:lez-2 unisce due lezioni, tag:cibo -tag:lez-1 prende il tema "cibo" escludendo la lezione 1.

Nota

Un mazzo filtrato "prende in prestito" le carte dal mazzo originale: i progressi restano collegati alla carta originale, quindi si può usare e cancellare senza perdere nulla.

05

Usare i diversi tipi di carta (template)

Ogni parola genera più carte (template), ad esempio "Russo → Italiano" e "Italiano → Russo". Per allenarsi solo in una direzione:

  1. Si va in Sfoglia.
  2. Si cerca card:"Russo → Italiano" (nome esatto del template) per vedere solo le carte in quella direzione. Si può combinare con un tag, es. card:"Russo → Italiano" tag:lez-3.
  3. Da lì si può creare un mazzo filtrato come al punto 04, oppure disattivare temporaneamente un template dalle impostazioni del Tipo di nota (Strumenti → Gestisci tipi di nota → Modelli), se non si vuole vedere affatto una delle due direzioni.

Se in futuro vengono aggiunti nuovi template (es. carte con audio o con frasi d'esempio), la stessa procedura funziona semplicemente usando il nuovo nome del template al posto di quello sopra.

06

Segnalare una modifica o un errore

Se si trova un errore (parola sbagliata, traduzione mancante, carta duplicata) o si vuole proporre un'aggiunta, il modo più semplice è mandare un messaggio con questi dati:

  • L'ID della carta, se lo si vede: in Anki → Sfoglia, selezionando la carta e guardando il campo ID (o aggiungendo la colonna "ID" alla lista). È il modo più preciso per identificarla, perché non cambia mai anche se la parola viene corretta.
  • In alternativa, la parola in russo (e in italiano, se nota) così com'è scritta nella carta.
  • Cosa non va o cosa si vorrebbe cambiare — es. "la traduzione è sbagliata, dovrebbe essere...", "manca la parola X", "serve un tag per il tema Y".
  • Se possibile, uno screenshot della carta in Anki: aiuta a trovarla subito.
Non serve modificare nulla in autonomia

Basta segnalare: la carta verrà corretta alla fonte e arriverà aggiornata nella prossima versione di russo.apkg — i progressi non si perdono.

07

Aggiornamenti e problemi frequenti

Quando arriva una nuova versione del file russo.apkg, la si importa seguendo di nuovo il passo 03. Le carte già esistenti vengono aggiornate senza cancellare i progressi.

Problemi frequenti

ProblemaCosa fare
Non compare nessuna carta dopo l'importSi controlla che il mazzo "Russo" sia visibile nella schermata principale di Anki
Le carte sono vuote o mostrano codici straniL'encoding non è corretto — si contatta chi ha mandato il file
L'app non riconosce il file .apkgCi si assicura che il file non sia stato rinominato durante il download
Non si riesce ad aprire il file .apkg su iPhoneSi prova da Safari invece che da altre app
